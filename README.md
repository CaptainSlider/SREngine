# SpaRcle Engine

## Used libraries
  * glm
  * OpenGL / GLFW / GLEW
  * Bullet
  * easy_profiler
  * Lua / LuaBridge
  * ImGUI

## Exists systems
  * Math
      * Quaternion
      * Vector3
      * Vector2
      * Matrix4x4
  * Graphics
      * Mesh render
      * GUI 
      * OBJ-loader
      * Texturing
      * Shaders
      * Materials
      * PostProcessing
        - Saturation
        - Bloom (Slow)
        - Color correction (Base)
  * Physics
      * Rigidbody (Base component)
  * Scripting
      * Base lua-api
  * Audio
  * ECS
  * Input system
  * Memory management
      * IResource - interface for control resource units
  * Debug and logging
 
## Editor
  * Grid
  * Manipulation tools
  * Inspector 
  * Hierarchy (70%)
  * Render settings
